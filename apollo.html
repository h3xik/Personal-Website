Projekt

Apollo

Tech stack:

SvelteKit, .Net Web Api, .Net Core

Software pro správu fitness centra a záznamů o klientech. Aplikace poskytuje moderní a intuitivní uživatelské rozhraní a tím zefektivňuje každodenní práci personálu. Apollo implementuje autentikaci pomocí bezkontaktních tagů a tím zjednodušuje život i klientům, kteří mohou jednoduše prokázat své členství. V plánu je i implementace klientského portálu, kde se zákazníci budou moct přihlásit na skupinové tréninky a další.
